/*
npm - global command, comes with node
npm --version

local dependency - use it only in this particular project
npm i <packageName>

global dependency - use it in any project
npm install -g <packageName>
sudo npm install -g <packageName> (mac)

package.json - manyfiest file (store important info about project/package), 3 ways:
    manual approach (create package.json in the root, create properties etc)
    npm init (step by step, press enter to skip)
    npm init -y (everything default)

In the scripts section, some commands like "start" you just need to do npm start,
but others like "dev", you need to do npm run dev
  "scripts": {
    "test": "echo \"Error: no test specified\" && exit 1",
    "start": "node app.js",
    "dev": "nodemon app.js"
  },
*/
